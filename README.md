# registro_habitos_salud
Registro de Hábitos de Salud Personal

Descripción
Proyecto final para el curso CIFO - Violeta. Uso de las librerías SQLite3, Numpy, TKinter, Pandas, Matplotlib.

El usuario podrá    1) Logear
                    2) Añadir Registros en su base de datos
                    3) Eliminar entrada de Registro
                    4) Mostrar historial de registros
                    5) Mostrar estadísticas Mensuales ( Promedios, máximos, mínimos )
                    6) Mostrar valores por debajo de ciertos límites.

La navegación por parte del usuario (GUI) se desarrollará mediante TKinter.
La permanencia de datos se desarrollará mediante SQLite3
